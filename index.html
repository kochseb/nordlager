4. Externe Ressourcen, Cookies und Datenverarbeitung auf unserer Website:

Unsere Website nutzt verschiedene externe Ressourcen und Technologien, um Funktionalität und Design zu gewährleisten. Hierbei kann es zur Übertragung von Daten an Drittanbieter kommen. Wir haben Maßnahmen ergriffen, um die Datenminimierung zu gewährleisten und weisen auf die Verantwortlichkeiten der jeweiligen Anbieter hin.

  • Content Delivery Networks (CDNs): Wir verwenden CDNs, um die Ladezeiten unserer Website zu optimieren und Inhalte wie Bilder, Skripte oder Stylesheets schneller bereitzustellen. Hierbei werden Daten an die CDN-Anbieter übertragen. Dies dient unserem berechtigten Interesse an einer schnellen und sicheren Bereitstellung unserer Website.
  • Google Fonts: Für eine ansprechende Darstellung der Schriftarten auf unserer Website nutzen wir Google Fonts. Beim Aufruf unserer Seite lädt Ihr Browser die benötigten Schriftarten in ihren Browser-Cache, um Texte und Schriftarten korrekt anzuzeigen. Dabei kann Ihre IP-Adresse an Google übertragen werden. Weitere Informationen zur Datenverarbeitung von Google Fonts finden Sie unter: Google Fonts Privacy FAQ und in der Datenschutzerklärung von Google: Google Datenschutzerklärung.
  • Tailwind CSS CDN: Für das Styling unserer Website nutzen wir das Tailwind CSS CDN. Hierbei werden CSS-Dateien von einem externen Server geladen. Es werden dabei keine personenbezogenen Daten von Tailwind CSS direkt erfasst, jedoch kann die IP-Adresse im Rahmen des Ladevorgangs an den CDN-Anbieter übertragen werden.
  • Font Awesome: Zur Darstellung von Icons auf unserer Website nutzen wir Font Awesome. Die Icons werden ebenfalls über ein CDN geladen, wodurch Ihre IP-Adresse an den Anbieter übermittelt werden kann. Weitere Informationen finden Sie in der Datenschutzerklärung von Font Awesome: Font Awesome Privacy Policy.
  • Externe Links: Unsere Website enthält Links zu externen Websites Dritter, auf deren Inhalte wir keinen Einfluss haben. Für die Inhalte der verlinkten Seiten ist stets der jeweilige Anbieter oder Betreiber der Seiten verantwortlich. Eine ständige inhaltliche Kontrolle der verlinkten Seiten ist ohne konkrete Anhaltspunkte einer Rechtsverletzung nicht zumutbar. Bei Bekanntwerden von Rechtsverletzungen werden wir derartige Links umgehend entfernen. Bitte beachten Sie die Datenschutzerklärungen der jeweiligen externen Websites.
